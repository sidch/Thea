Detailed Description

template<typename T>
class Thea::Algorithms::FixedPointIterator< T >

Iterates repeatedly over a single object.

If T is a non-const type, this object can be modified (e.g. if it expresses a count), though this feature is probably of limited value, and is especially brittle if copies of the iterator are involved. This class satisfies the RandomAccessIterator specification. However, some functions such as the <, > <=, >= etc operators are inconsistent and should not be used.
