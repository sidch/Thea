Compute the PCA axes of a set of N-D objects.

ObjectInputIterator must dereference to type T or pointer-to-T.

Parameters
beginIterator pointing to the first object in the set.
endIterator pointing to one position beyond the last object in the set.
eigenvaluesThe eigenvalues of the covariance matrix, ordered from largest to smallest.
eigenvectorsThe unit-length eigenvectors of the covariance matrix, ordered from largest to smallest eigenvalue.
centroidIf non-null, used to return the centroid of the objects, which is computed during PCA.

Compute the PCA axes of a set of N-D objects with per-object weights.

ObjectInputIterator must dereference to type T or pointer-to-T. WeightInputIterator must dereference to a scalar.

Parameters
beginIterator pointing to the first object in the set.
endIterator pointing to one position beyond the last object in the set.
weights_beginIterator pointing to the weight of the first object.
eigenvaluesThe eigenvalues of the weighted covariance matrix, ordered from largest to smallest.
eigenvectorsThe unit-length eigenvectors of the weighted covariance matrix, ordered from largest to smallest eigenvalue.
centroidIf non-null, used to return the weighted centroid of the objects, which is computed during PCA.
