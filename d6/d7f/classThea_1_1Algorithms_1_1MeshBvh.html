Parameters

dist_boundUpper bound on the distance between any pair of points considered. Ignored if negative.

It should have a bool operator()(q, e) const function that returns true if the two objects q and e are compatible with each other, where q is (i) an element of QueryT if the latter is a ProximityQueryStructureN, or (ii) query otherwise, or (iii) a TransformedObject wrapping either of these; and e is (i) an element of this structure, or (ii) a TransformedObject wrapping it.

Recursively construct a (sub-)kd-tree.

If save_memory is true, element indices for this subtree are assumed to be in a block at the end of main_index_pool. They are subsequently moved to arrays associated with the leaves, allocated in leaf_index_pool. They are deleted from main_index_pool when this function exits, thus deallocating index arrays held by internal nodes. If save_memory is false, each node maintains its own list of all elements in its subtree.

Note
To use this function with a node accessed via getRoot(), you'll have to const_cast it to a non-const type first.

Get the k elements closest to a query object.

The returned elements are placed in a set of bounded size (k). The template type BoundedNeighborPairSetT should typically be BoundedSortedArray<NeighborPair> or BoundedSortedArrayN<k, NeighborPair> if only a few neighbors are requested.

Parameters
queryQuery object. BoundedTraitsN<QueryT, N, ScalarT> must be defined.
k_closest_pairsThe k (or fewer) nearest neighbors are placed here.
dist_boundUpper bound on the distance between any pair of points considered. Ignored if negative.
compatibilityA functor that checks if two elements being considered as near neighbors are compatible with each other or not. It should have a bool operator()(q, e) const function that returns true if the two objects q and e are compatible with each other, where q is (i) an element of QueryT if the latter is a ProximityQueryStructureN, or (ii) query otherwise, or (iii) a TransformedObject wrapping either of these; and e is (i) an element of this structure, or (ii) a TransformedObject wrapping it.
