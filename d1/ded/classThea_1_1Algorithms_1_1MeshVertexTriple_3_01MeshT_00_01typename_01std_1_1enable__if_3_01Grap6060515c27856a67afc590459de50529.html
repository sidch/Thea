Detailed Description

template<typename MeshT>
class Thea::Algorithms::MeshVertexTriple< MeshT, typename std::enable_if< Graphics::IsDisplayMesh< MeshT >::value >::type >

A set of three vertices of a single face of a display mesh.

If the mesh vertices change, so does this triplet.
