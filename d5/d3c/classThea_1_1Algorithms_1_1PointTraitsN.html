Detailed Description

template<typename T, int N, typename ScalarT = Real, typename Enable = void>
class Thea::Algorithms::PointTraitsN< T, N, ScalarT, Enable >

Traits for an object which can be identified with a single point in N-space.

If IsPointN<T>::value is false, then the getPosition() function returns some arbitrary value by default. The default implementation for IsPointN<T>::value == true handles Vector and types implicitly convertible to it. Specialize as needed for other classes.
