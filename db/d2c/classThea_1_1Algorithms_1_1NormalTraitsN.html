Detailed Description

template<typename T, int N, typename ScalarT = Real, typename Enable = void>
class Thea::Algorithms::NormalTraitsN< T, N, ScalarT, Enable >

Traits for an object which has an N-D normal vector.

If HasNormalN<T>::value is false, then the getNormal() function returns some arbitrary value by default. The default implementation for HasNormalN<T>::value == true assumes the type T has a T::getNormal() member function. Specialize as needed for other classes.
